# UML (Nnified Modeling Language)

## UML 이란 ?

### UML
* 객체 지향 분석/설계 산물(artifacts)을 위한, 표준화된 notation
* UML은 모델링 언어의 통합을 위한 표준
* UML은 S/W를 시각화, 명세화, 문서화하기 위한 언어
